### создание виртуального окружения
```
python3 -m venv venv 
```
### активация вирутального окружения
```
source venv/bin/activate
```
### деактивация вирутального окружения
```
deactivate
```
### установка пакетов
```
pip install <название пакета>
```
### проверка списка пакетов
```
pip freeze / pip list
```
